# Exovia NeuroCanvas — guía fácil

Esta guía está pensada para personas que no usan herramientas técnicas habitualmente, incluidas personas mayores.

## Qué hace

NeuroCanvas ayuda a ordenar información y volver a encontrarla.

Puede servir para:

- estudiar;
- organizar recetas;
- guardar historias familiares;
- revisar documentos;
- preparar clases;
- investigar un tema;
- ordenar notas de trabajo;
- comparar ideas;
- conservar decisiones y sus fuentes.

La idea central es sencilla:

```text
Ponés información
→ aparece un mapa
→ elegís una idea
→ ves la fuente
→ hacés una pregunta
→ guardás el resultado
```

## Vista simple

En la parte superior aparece el botón:

```text
Simple view
```

Al activarlo:

- el texto se hace más grande;
- los botones tienen más tamaño;
- se ocultan opciones avanzadas;
- quedan visibles las tareas principales;
- la aplicación recuerda esta preferencia.

Para volver a ver todas las opciones, pulsar:

```text
Standard view
```

## Ayuda paso a paso

Pulsar:

```text
Guide me
```

La guía muestra cuatro pasos:

1. crear o importar información;
2. seleccionar una idea;
3. hacer una pregunta;
4. guardar el trabajo.

Cada paso resalta el botón o zona que corresponde.

## Primer uso recomendado

1. Pulsar `Simple view`.
2. Pulsar `Guide me`.
3. Elegir `Create a sample map`.
4. Pulsar un círculo del mapa.
5. Leer el panel derecho.
6. Abrir `Answer & Audit`.
7. Escribir una pregunta común.
8. Guardar con `Save`.
9. Crear una copia con `Export`.

## Ejemplos útiles

### Recetas

Pegar recetas, ingredientes y notas. Después preguntar:

```text
¿Qué recetas usan papa?
```

### Historia familiar

Guardar fechas, nombres, lugares y anécdotas. Después buscar una persona o acontecimiento.

### Trabajo

Importar notas de reuniones y revisar qué decisión se tomó y de dónde salió.

### Estudio

Pegar apuntes y preguntar por conceptos. Luego volver a la evidencia original.

### Salud personal

Puede organizar información general o preguntas para llevar al médico, pero no reemplaza asesoramiento médico profesional.

## Seguridad

- La información local no se publica automáticamente.
- Guardar con frecuencia.
- Usar `Export` para crear una copia.
- No incluir contraseñas ni claves privadas.
- No borrar datos del navegador sin exportar antes.

## Cuando algo no se entiende

Pulsar:

```text
What problem does this solve?
```

Esa explicación resume el producto y su utilidad.

También se puede volver a abrir `Guide me` en cualquier momento.
