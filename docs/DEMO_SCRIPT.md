# Three-Minute Demo Script

## 0:00–0:25 — The problem

“Today we navigate the physical world with maps, but we still navigate knowledge as flat pages and endless chat history. Exovia NeuroCanvas turns complex information into a territory.”

Show the empty landing screen and the tagline.

## 0:25–0:55 — Create the map

Click **Load demo**. Explain that the MVP works offline and requires no API credits. Point out the corpus root, topic clusters and source fragments.

## 0:55–1:25 — Navigate knowledge

Pan and zoom across the canvas. Switch from **Neural** to **Tree** view.

Explain:

- Tree view reveals hierarchy.
- Neural view reveals cross-topic relationships.
- The original text remains preserved behind every node.

## 1:25–2:05 — Zoom to Answer

Search for `privacy`.

The camera moves to the strongest matching node and highlights related evidence. Open the node in the inspector and show the exact source text and semantic path.

Say: “The interface does not ask users to trust an abstract summary. It takes them directly to the evidence.”

## 2:05–2:35 — New content

Click **Paste text**, paste a short document, and build a new map. Demonstrate that processing happens locally. Export the resulting project as JSON.

## 2:35–3:00 — Vision

“The current build proves the interaction offline. The next provider adds OpenAI embeddings, structured summaries and grounded answers through a secure server. The long-term vision is a universal visual memory for documents, code, research and conversations.”

End with:

**Knowledge should be navigated, not merely scrolled.**
