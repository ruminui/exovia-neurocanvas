# OpenAI Build Week Submission

## Project name

Exovia NeuroCanvas

## Creator

Luciano — Exovia

## Tagline

Navigate knowledge like a living neural map.

## One-line description

An offline-first infinite canvas that transforms long documents, conversations and code into a navigable semantic tree and neural graph with exact source evidence behind every node.

## About the project

Today we navigate the physical world with maps, but we still navigate knowledge as flat pages and endless chat history. Exovia NeuroCanvas turns complex information into a living visual territory.

Users begin with a global view of a corpus, zoom into themes and subthemes, discover semantic relationships, and finally inspect the exact original source text. The project combines local text processing, hierarchical organization, graph visualization, progressive detail and a distinctive “Zoom to Answer” interaction.

The current MVP is offline-first and runs without API credits. It performs local chunking, keyword extraction, clustering, similarity scoring, tree generation, graph generation, search and camera focus. The architecture is designed so OpenAI embeddings and grounded Responses API answers can be added as an optional secure server-side provider.

## Problem

Long documents and chat histories are difficult to understand as systems. Search finds strings, but it does not reveal structure. Summaries reduce detail, but they can hide evidence. Static mind maps require manual organization and do not scale.

## Solution

NeuroCanvas preserves the source while creating multiple synchronized views:

- a hierarchical tree for structure;
- a neural graph for cross-topic relationships;
- an infinite canvas for spatial navigation;
- an evidence inspector for verification;
- Zoom to Answer for moving directly from a query to relevant source nodes.

## Innovation

The visualization is not treated as the memory itself. It is a doorway into structured, verifiable memory. This avoids the fidelity problems of storing microscopic text only as raster images while retaining the powerful idea of infinite visual zoom.

## Built with

HTML5 Canvas, JavaScript, CSS, local semantic heuristics, JSON project persistence, and an offline-first architecture prepared for optional OpenAI integration.

## Impact

Potential applications include organizational memory, education, research, codebase exploration, long-term conversation memory, consulting knowledge bases and visual document analysis.

## Demo flow

1. Open the application and load the built-in demo.
2. Pan and zoom through the neural map.
3. Switch between Neural and Tree views.
4. Search for “privacy” or “Zoom to Answer.”
5. Watch the camera focus on the strongest evidence.
6. Open a node and show the exact source text.
7. Paste a new document and generate a new map locally.
8. Export the project as JSON.

## Key phrase

**Knowledge should be navigated, not merely scrolled.**
