Exovia NeuroCanvas ChatGPT App — Privacy

Default behavior: the MCP service processes submitted text in memory to produce the requested result. This reference implementation does not create user accounts, persist project content, call an external AI provider, sell personal data, or execute external actions.

Operational infrastructure may process standard request metadata required to deliver and secure the service, such as timestamps, network information, response status and error details. Application error logs are designed not to intentionally record submitted project content.

The tools use deterministic heuristics. Users should avoid submitting credentials or unnecessary personal information and should review outputs before relying on them.

Human control

Reliability scores, comparisons, maps and Proof Packs are decision-support artifacts. They do not replace live source verification, professional advice or human approval.
